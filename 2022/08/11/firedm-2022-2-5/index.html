FireDM v2022.2.5

FireDM could somewhat be a handy media downloader. It supports all functions of YouTube-dl and has a GUI, which is newbie-friendly.

But the main repo seemed to be taken down by GitHub. The latest version “2022.4.14 Frozen” doesn’t work anymore. However, we can use version “2022.2.5”

Archive for Windows users

How to use on Windows

  1. Unzip FireDM.full.for.Windows.64.bit.7z

  2. Click FireDM-GUI.exe

  3. Choose the desired formats before proceeding with the download

  4. New files will show up in the folder you chose. Enjoy!

Note: Add firedm.exe to PATH for convenience’s sake. Run firedm in Command Prompt by typing firedm next afterward.

For Linux and Mac users, try typing these commands in a terminal:

After the installation, simply type firedm to launch the GUI.
